Setup
-----------

- Update database.yml with username / password of your database
- Run 'rake db:create'
- Run 'rake db:seed'  
- Run 'rails s'

Ruby on Rails
-------------

This application requires:

- Ruby 2.4.2
- Rails 5.2.0
